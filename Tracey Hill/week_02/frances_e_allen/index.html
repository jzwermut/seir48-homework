Frances 'Fran' Allen

American Computer Scientist

Key research areas:

  • Experimental Compiling System (ECS) described here represented a new compiler construction methodology. This type of compiler meant that computer hardware could run more than one source programming language contributing to the research toward a more general use case. This research contributed to understandings that enabled generalised types of computer hardware to support generalised modern operating systems and programs we use today. Read more in her article "The Experimental Compiling System"
  • Allen was assigned to the Harvest project for code breaking, a cryptanalysis research centre gathering intelligence, with the National Security Agency in 1959 and was assigned to work on a programming language called Alpha. Alpha influenced the design of QUEL back in th 70's, which was a precursor to SQL that was the foundation of IBM's first commercial relational database product. Read more in about the Harvest project here.
